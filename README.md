# Plume

Cross-browser Micropub client extension. Post to your IndieWeb-compatible blog from any page.

**Status:** v0.1 — pre-release. See `docs/superpowers/specs/2026-05-17-plume-micropub-extension-design.md` for design.

## Development

```bash
bun install
bun run dev          # Chrome
bun run dev:firefox  # Firefox
bun test
```
